## 作品說明
「COFFSEEKER」是一家精選來自世界各地咖啡豆的品牌。我們深知現代人對咖啡風味的精緻要求，因此精心挑選各種不同的咖啡產品，希望使用者能在這裡找到屬於自己獨特的咖啡饗宴。
此專案為COFFSEEKER的初期後台。



## 我負責的部分

該專題的課程管理後台頁面以及檔案庫建置。

使用PHP、phpMyAdmin、 Bootstrap建置課程管理後台

具備功能：新增 / 修改 / 刪除 / 排序 / 分頁/ 搜尋/ 多重篩選/ 上下架功能
